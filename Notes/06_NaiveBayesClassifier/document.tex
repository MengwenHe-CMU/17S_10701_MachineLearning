Optimal Classification (Bayes classifier)

$$f^* = \arg\min_f P(f(X)\neq Y)$$

Bayes rule $P(Y|X) \propto P(X|Y)P(Y)$
\begin{itemize}
	\item $P(X|Y)$: class conditional distribution
	\item $P(Y)$: class probability
\end{itemize}
